Учебный офис

Навигация и действия

Слева находятся служебные панели. Основной сценарий работы расположен в центре страницы в формате чата.

Система

Ответы строятся по найденным фрагментам. Если LLM не настроен, сервис всё равно покажет найденные материалы.

Действия

Открыть админ-панель

Переиндексация и управление документами перенесены в административную панель.

Чат

Чат по базе документов

Интерфейс выглядит как чат, но не сохраняет контекст диалога: каждый вопрос отправляется как отдельный запрос к базе документов.

Готов Источники прикрепляются к каждому ответу отдельно.
Ассистент

Задайте вопрос по локальной базе документов. Для надёжности формулируйте каждый запрос полностью, без расчёта на память предыдущих сообщений.

Контекст не переносится между сообщениями. Если вопрос зависит от прошлого ответа, повторите нужные детали в новом сообщении.

Каталог

Документы базы

Здесь перечислены документы, по которым строятся ответы. Для управления составом базы используйте админ-панель.
